## Post 2: Technical post (post 3 days after Post 1)

How I built a production-grade AI platform as a BTech student with Rs.0 budget:

Tech stack:
- Next.js 14 + TypeScript (frontend, Vercel - free)
- Python FastAPI (backend, Render - free)
- OpenRouter -> Gemini 2.0 Flash (AI - free tier)
- SQLite -> PostgreSQL-ready (database - free)

Engineering features shipped:
- Real-time SSE streaming with fallback engine
- Response caching with TTL and LRU eviction
- Per-IP rate limiting
- 35 backend unit tests, CI green on every push
- PDF export, keyboard shortcuts, semantic search

3 months. Solo. Still in BTech. Live at scholr-coral.vercel.app

What I learned: shipping > perfecting. Get it live, get feedback, iterate.
