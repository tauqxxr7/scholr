# Screenshots

This folder contains the product screenshots used in the top-level README.

Current files:
- `landing.png`
- `research-workspace.png`
- `research-output.png`
- `notes-output.png`
- `doubt-output.png`

Planned asset:
- `scholr-demo.gif`

The GIF is still pending because it was not generated automatically from this environment. Until then, the README uses the live screenshots as the demo preview.

Suggested low-friction GIF workflow:
1. Record a 10-20 second flow: landing -> research -> notes/doubt output.
2. Export to MP4 from your screen recorder.
3. Convert to `screenshots/scholr-demo.gif` with a lightweight tool like ScreenToGif or ffmpeg.
4. Keep the GIF short and under a practical GitHub-friendly file size.
