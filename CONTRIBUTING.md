# Contributing to Scholr

Thank you for your interest in Scholr.

## Current status
Scholr is a solo project by Tauqeer Bharde, currently in active development. The codebase is open for review and feedback.

## How to give feedback
- Open an Issue describing what you found
- Use the /feedback page on the live product for product feedback
- Email tauqeerplayer@gmail.com for direct feedback

## Local setup
See README.md for the 60-second local setup guide.

## Code style
- Backend: Python, follow existing FastAPI patterns, no new dependencies without discussion
- Frontend: TypeScript, Tailwind CSS, shadcn/ui components, Next.js App Router

## Pull requests
Not actively accepting PRs at this stage. Raise an Issue first.
